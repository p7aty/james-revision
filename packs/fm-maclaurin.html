Maclaurin Series

Power series expansions · Standard results · Range of validity

Session plan — RED topic
  1. 10 min — Watch ExamSolutions Maclaurin intro (or equivalent).
  2. 15 min — Learn the general formula and the 4 standard series cold.
  3. 30 min — Work through the three examples below.
  4. 20 min — Practice problems at bottom.

These four series are in the formula booklet, but you still need to know them cold — exam questions frequently test manipulating and combining them under time pressure, and flipping to the booklet every time costs marks.

The core formula

A Maclaurin series is the Taylor series of a function expanded around x = 0. It expresses f(x) as an infinite polynomial in powers of x.

f(x) = f(0) + f'(0)·x + f''(0)
2!·x2 + f'''(0)
3!·x3 + ... + f(n)(0)
n!·xn + ...

To build a Maclaurin series from scratch:

  1. Differentiate f(x) several times.
  2. Evaluate each derivative at x = 0.
  3. Divide the nth derivative at 0 by n! and multiply by xn.
  4. Sum the terms.
Requirement for the series to exist: all derivatives f(n)(0) must exist (be finite). That's why you can't Maclaurin-expand ln(x) at x = 0 — ln(0) is undefined. Instead, expand ln(1 + x) which is defined and well-behaved at x = 0.

The four standard series — memorise cold

1. Exponential: ex

ex = 1 + x + x2
2! + x3
3! + ... + xn
n! + ...

Why: every derivative of ex is ex, which equals 1 at x = 0. So every coefficient is 1/n!

Range of validity: all real x. Converges everywhere.

2. Logarithm: ln(1 + x)

ln(1 + x) = x − x2
2 + x3
3 − x4
4 + ... + (−1)n+1xn
n + ...

Key features: alternating signs starting positive; denominators are just n (no factorial). Much slower convergence than ex.

Range of validity: −1 < x ≤ 1. Critical point: it FAILS for x ≤ −1 (where ln is undefined) and is only borderline-valid at x = 1.

3. Sine: sin x

sin x = x − x3
3! + x5
5! − x7
7! + ... + (−1)nx2n+1
(2n+1)! + ...

Key features: only odd powers of x (because sin is odd). Alternating signs. Denominators are odd factorials.

Range of validity: all real x. Converges everywhere.

4. Cosine: cos x

cos x = 1 − x2
2! + x4
4! − x6
6! + ... + (−1)nx2n
(2n)! + ...

Key features: only even powers of x (cos is even). Alternating signs starting from +1. Denominators are even factorials.

Range of validity: all real x. Converges everywhere.

Pattern to memorise:

  • sin and cos alternate odd/even powers.
  • cos = derivative of sin shifted — and you can check this: differentiate the sin series term-by-term to get the cos series.
  • ex, sin, cos all have factorial denominators. ln(1+x) does NOT (just n).
  • All three trig and exp series converge for ALL x. Only ln is restricted.

Worked example 1 — derive a series from scratch

Q: Find the first four non-zero terms of the Maclaurin series for f(x) = (1 + x)1/2.

Method. Differentiate repeatedly, evaluate at x = 0.

nf(n)(x)f(n)(0)Term
0(1+x)1/211
1½(1+x)−1/2½(½)x = x/2
2−¼(1+x)−3/2−¼(−¼/2!)x² = −x²/8
3⅜(1+x)−5/2⅜(⅜/3!)x³ = x³/16
4−15⁄16(1+x)−7/2−15⁄16(−15⁄16/4!)x⁴ = −5x⁴/128

Answer: (1 + x)1/2 ≈ 1 + x/2 − x²/8 + x³/16 − 5x⁴/128 + ...

Valid for |x| < 1 (binomial-type series).

Worked example 2 — use standard series

Q: Find the Maclaurin series for f(x) = e2x as far as the term in x⁴.

Method. Don't differentiate from scratch — substitute into the known ex series.

ex = 1 + x + x²/2! + x³/3! + x⁴/4! + ...

Replace x with 2x:

e2x = 1 + (2x) + (2x)²/2! + (2x)³/3! + (2x)⁴/4! + ...

     = 1 + 2x + 4x²/2 + 8x³/6 + 16x⁴/24 + ...

     = 1 + 2x + 2x² + 4x³/3 + 2x⁴/3 + ...

Valid for all real x.

Worked example 3 — combine series

Q: Find the Maclaurin series for f(x) = ex·sin x up to the term in x⁴.

Method. Multiply the two known series, keeping terms only up to x⁴.

ex = 1 + x + x²/2 + x³/6 + x⁴/24 + ...

sin x = x − x³/6 + x⁵/120 − ...

Multiply, only keeping terms whose combined power ≤ 4:

1·(x − x³/6) + x·(x − x³/6) + (x²/2)·x + (x³/6)·x + ...

= x − x³/6 + x² − x⁴/6 + x³/2 + x⁴/6

= x + x² + (−x³/6 + x³/2) + (−x⁴/6 + x⁴/6)

= x + x² + x³(−1/6 + 3/6) + 0

= x + x² + x³/3 + ... (x⁴ coefficient is 0 in this case)

Careful: we need terms up to x⁴, so continue to x⁵ products if x⁴ coefficient cancels to zero. Extending: next nonzero term comes from (x²/2)(x³)... actually x⁵ products can't contribute to x⁴.

Using Maclaurin to approximate values

Q: Use the Maclaurin series for cos x to estimate cos(0.1) to 5 decimal places.

cos x = 1 − x²/2! + x⁴/4! − ...

cos(0.1) = 1 − 0.01/2 + 0.0001/24 − ...

     = 1 − 0.005 + 0.00000417 − ...

     ≈ 0.99500

Calculator check: cos(0.1) = 0.99500... ✓. The approximation is excellent because x = 0.1 is small — the next term (x⁶/6!) is ≈ 10−9, negligible.

Range of validity — why it matters

Maclaurin series don't always converge to the function. The range where they do is the radius of convergence.

FunctionValid for
ex, sin x, cos xAll real x
ln(1 + x)−1 < x ≤ 1
(1 + x)n (non-integer n)|x| < 1
1/(1 − x) = 1 + x + x² + ...|x| < 1

Key intuition: the series for f(x) will usually be valid within the nearest distance from 0 to any point where f is undefined or singular. ln(1+x) has a problem at x = −1 (ln 0), so valid radius is 1.

Exam warning. If the question gives x = 2 and asks you to use ln(1+x) series, the series won't converge. State this and refuse — examiners are testing whether you check validity.

Two-minute summary

  • Maclaurin formula: f(x) = Σ f(n)(0) xn / n!
  • ex: 1 + x + x²/2! + x³/3! + ... (all x)
  • ln(1+x): x − x²/2 + x³/3 − x⁴/4 + ... (−1 < x ≤ 1)
  • sin x: x − x³/3! + x⁵/5! − ... (all x)
  • cos x: 1 − x²/2! + x⁴/4! − ... (all x)
  • Strategies: substitute (e2x, sin(x²)); multiply two series (ex sin x); differentiate or integrate term-by-term.
  • Always check: does the x-value lie in the range of validity?

Practice problems

  1. Write out the Maclaurin series for ex, ln(1+x), sin x, cos x from memory. Include range of validity for each.
  2. Find the Maclaurin series for e−x as far as x⁴.
  3. Find the Maclaurin series for sin(2x) as far as x⁵.
  4. Find the series for ln(1 − x) and state its range of validity.
  5. Find the Maclaurin series for cos(x²) as far as x⁸.
  6. Use the first three non-zero terms of the cos x series to estimate cos(0.2). Compare with your calculator.
  7. Find the first four non-zero terms of the Maclaurin series for (1 + x)−1/2.
  8. Derive the first four non-zero terms of the Maclaurin series for tan x. (Hint: use sin/cos and long division or power series division.)

Quick answers: Q2: 1 − x + x²/2 − x³/6 + x⁴/24. Q3: 2x − 4x³/3 + 4x⁵/15. Q4: −x − x²/2 − x³/3 − ... (valid for −1 ≤ x < 1). Q5: 1 − x⁴/2 + x⁸/24. Q6: cos(0.2) ≈ 1 − 0.02 + 0.0000667 ≈ 0.98007. Calculator: 0.98007. Q7: 1 − x/2 + 3x²/8 − 5x³/16 + 35x⁴/128. Q8: tan x ≈ x + x³/3 + 2x⁵/15 + 17x⁷/315.
