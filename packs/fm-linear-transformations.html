Linear Transformations

2D and 3D matrix transformations · Invariant points and lines

Session plan — this is a RED topic
  1. 15 min — Watch an ExamSolutions video on matrix transformations (or similar).
  2. 25 min — Learn the standard 2D matrices below cold — the ones listed in the formula booklet are NOT enough, you need rotation and reflection matrices at memory level.
  3. 30 min — Work through the three worked examples below.
  4. 20 min — Practice questions at the bottom.

If stuck: book a tutor session this week. This topic is foundational for invariant lines, eigenvectors (later), and shows up in most FM Paper 1s.

Core idea

A 2×2 matrix acting on a position vector (x, y)ᵀ transforms the point to a new location (x', y')ᵀ. Every 2D linear transformation corresponds to a unique 2×2 matrix, and every 2×2 matrix corresponds to some linear transformation.

The trick to finding any transformation matrix: work out where the standard basis vectors go.

  • The first column of the matrix tells you where (1, 0) — i.e. the x-axis unit vector i — gets mapped to.
  • The second column tells you where (0, 1) — i.e. the y-axis unit vector j — gets mapped to.

Draw a sketch of where i and j end up, read off the columns. Done.

Composite transformations

If transformation T₁ is applied first, followed by T₂, the combined matrix is:

M = T₂ T₁

The later transformation goes on the LEFT. This trips up everyone. The matrix closest to the vector acts first.

Matrix multiplication is not commutative — the order matters. T₁T₂ ≠ T₂T₁ in general.

Invariant points and invariant lines

Critical distinction — write this out until you can't forget it

  • Invariant point — maps to itself: Tp = p. The origin is always invariant under any linear transformation.
  • Line of invariant points — every point on the line is invariant. Each point stays exactly where it was.
  • Invariant line — the line maps to itself as a whole, but individual points can slide along it. A weaker condition than "line of invariant points".

A line of invariant points IS an invariant line, but not vice versa. Reflections and shears typically have lines of invariant points. Stretches and enlargements typically have invariant lines where points slide along.

Finding invariant points

Solve Mp = p, i.e. (M − I)p = 0. This is a homogeneous system. If det(M − I) ≠ 0, only the origin is invariant. If det(M − I) = 0, there's a line of invariant points.

Finding invariant lines through the origin

Assume a line y = mx. A general point (x, mx) must map to another point on the same line — i.e. y' = mx' after the transformation. Substitute into the transformation equations, equate, and solve for m. Usually get a quadratic giving up to two invariant lines.

The determinant — area and volume scale factor

  • det(M) = area scale factor (in 2D). An enlargement by k has |det M| = k².
  • det(M) > 0 — orientation preserved (rotations, enlargements).
  • det(M) < 0 — orientation reversed (any reflection).
  • det(M) = 0 — the matrix is singular. The transformation squashes 2D space onto a line or point — information is lost.
  • In 3D: det(M) = volume scale factor.

Worked example 1 — describe a transformation

Q: Describe the transformation represented by

0−1
−10
.

Method 1: where does i go? i = (1, 0). First column of matrix is (0, −1) — so i → (0, −1), i.e. the negative y-axis.

Where does j go? j = (0, 1). Second column is (−1, 0) — so j → (−1, 0), the negative x-axis.

Sketch: the unit square has flipped to the third quadrant in a reflected way (not just rotated). det = (0)(0) − (−1)(−1) = −1, so it's orientation-reversing. So a reflection.

In what line? The line y = −x swaps i ↔ −j and j ↔ −i. Yes. Answer: reflection in the line y = −x.

Worked example 3 — find an invariant line

Q: Find the invariant lines (through the origin) of

34
9−2
.

Method. Take a general point (x, mx) on the line y = mx. After transformation:

x' = 3x + 4(mx) = (3 + 4m)x

y' = 9x + (−2)(mx) = (9 − 2m)x

For the image to still be on y = mx, we need y' = mx':

(9 − 2m)x = m(3 + 4m)x

9 − 2m = 3m + 4m²

4m² + 5m − 9 = 0

(4m + 9)(m − 1) = 0

m = 1 or m = −9/4

Answer: y = x and y = −9x/4 are the invariant lines.

(Also check if the y-axis itself is invariant by trying a point on it — if (0, y) maps to (4y, −2y), not on y-axis, so y-axis is not invariant here.)

Two-minute summary

  • Find the matrix: column 1 is where (1,0) goes; column 2 is where (0,1) goes.
  • Composite: later transformation on the LEFT. M = T₂T₁ applies T₁ first.
  • Invariant point: Mp = p. Origin always invariant.
  • Line of invariant points: every point stays (reflections, shears).
  • Invariant line: line maps to itself but points can slide (stretches).
  • det(M): area (2D) or volume (3D) scale factor. Negative = reflection. Zero = singular.
  • Key matrices to memorise: reflections in x-axis, y-axis, y=x, y=−x; 90° rotation AC; general rotation [cos, -sin / sin, cos]; enlargement by k.
