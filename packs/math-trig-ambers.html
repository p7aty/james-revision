E2 — Small angle approximations

The three approximations — memorise

For small x (in radians):

sin x ≈ x

cos x ≈ 1 − x²/2

tan x ≈ x

Why — these are the first non-trivial terms of the Maclaurin series for each function. For |x| small, higher-order terms become negligible.

Critical: the approximations only work in radians, not degrees. Questions always use radians. If a question says "x is small and in degrees", convert first.

Accuracy — when does it break?

Rough guide:

  • For |x| < 0.1 rad (~5.7°), all three approximations are accurate to ~4 decimal places.
  • For |x| < 0.5 rad (~28°), sin and tan diverge from x by a few percent.
  • For |x| around 1 rad, errors exceed 10% — don't use.

AQA questions will always give you an "x is small" setup so the approximations apply. Never second-guess this in the exam.

Key techniques

Technique 1 — direct substitution.

"Given x is small, find approximate value of sin(3x) cos(2x)."

sin(3x) ≈ 3x (replace x with 3x in sin x ≈ x)

cos(2x) ≈ 1 − (2x)²/2 = 1 − 2x²

Product ≈ 3x · (1 − 2x²) = 3x − 6x³

For "small" angles, keep terms up to the highest power that's still meaningful (usually x² or x³).

Technique 2 — first order vs second order.

cos x has no linear term (it starts 1 − x²/2 + ...), so you sometimes need cos ≈ 1 for very rough work, or cos ≈ 1 − x²/2 when x² matters.

Read the question: "Find the first two non-zero terms..." = you need the quadratic; "Approximate to first order" = linear only.

Five practice questions

  1. Given x is small, find an approximation for sin(4x) + cos(3x) as a polynomial in x.
  2. Use small-angle approximations to find limx→0 (sin 3x) / (tan 5x).
  3. Given x is small, show that (1 + cos x) / (1 − cos x) ≈ 4/x².
  4. Find an approximation for (sin x · cos x) / x³, valid for x small and non-zero.
  5. Use a small-angle approximation to estimate sin(0.02). Compare with your calculator.

Quick answers: Q1: sin(4x) ≈ 4x, cos(3x) ≈ 1 − 9x²/2. Sum ≈ 1 + 4x − 9x²/2. Q2: sin(3x)/tan(5x) ≈ 3x/5x = 3/5. Q3: 1 + cos x ≈ 2, 1 − cos x ≈ x²/2. Ratio = 2 ÷ (x²/2) = 4/x². ✓ Q4: sin x · cos x ≈ x · 1 = x (first order). So (sin x cos x)/x³ ≈ 1/x². (Keep terms as needed.) Q5: sin(0.02) ≈ 0.02. Calculator: 0.01999866... — error ~10⁻⁶.

E4 — R sin(x + α) form

The core identity

a sin x + b cos x ≡ R sin(x + α)

where R = √(a² + b²) and tan α = b/a

Also commonly used:

  • a sin x − b cos x ≡ R sin(x − α)
  • a cos x + b sin x ≡ R cos(x − α)
  • a cos x − b sin x ≡ R cos(x + α)

Same R = √(a² + b²); α is always a positive acute angle in the first quadrant. Choose the identity that gives you a positive α.

How to derive

Expand R sin(x + α) using the compound angle formula:

R sin(x + α) = R sin x cos α + R cos x sin α

Compare with a sin x + b cos x:

  • R cos α = a
  • R sin α = b

Square and add: R² cos²α + R² sin²α = a² + b² → R² = a² + b² → R = √(a² + b²).

Divide: tan α = b/a.

Why use this form

  1. Finding max/min of a sin x + b cos x. The expression ≡ R sin(x + α) has maximum value R (when sin = 1) at x = π/2 − α, and minimum −R at x = 3π/2 − α.
  2. Solving equations like a sin x + b cos x = c. Convert to R sin(x + α) = c, then x + α = arcsin(c/R) + more roots. Much simpler than tackling a sin + b cos directly.
  3. Modelling waves. Any sinusoidal oscillation with amplitude and phase can be written in R sin(ωt + φ) form.

Worked example — the standard exam question

Q: Express 3 sin x + 4 cos x in the form R sin(x + α), where R > 0 and 0 < α < π/2. Hence solve 3 sin x + 4 cos x = 2 for 0 ≤ x < 2π.

Part 1 — convert:

R = √(3² + 4²) = √25 = 5

tan α = 4/3, so α = arctan(4/3) ≈ 0.927 rad

So 3 sin x + 4 cos x ≡ 5 sin(x + 0.927)

Part 2 — solve:

5 sin(x + 0.927) = 2

sin(x + 0.927) = 0.4

x + 0.927 = arcsin(0.4) = 0.412 rad   OR   x + 0.927 = π − 0.412 = 2.730 rad

So x = 0.412 − 0.927 = −0.515 (not in range; add 2π: 5.768)

Or x = 2.730 − 0.927 = 1.803

Solutions: x = 1.803 rad and x = 5.768 rad (to 3 dp)

Trap to avoid. When solving, remember sin θ = k has TWO solutions per 2π interval: θ = arcsin(k) and θ = π − arcsin(k). Students commonly forget the second. Always consider both.

Five practice questions

  1. Express 5 sin x − 12 cos x in the form R sin(x − α), R > 0, 0 < α < π/2.
  2. Find the maximum and minimum values of 6 cos x + 8 sin x and the value of x ∈ [0, 2π) where each occurs.
  3. Solve √3 sin x + cos x = 1 for 0 ≤ x < 2π.
  4. A small object oscillates so that its displacement from a fixed point at time t seconds is given by y = 4 sin(2t) + 3 cos(2t) metres. Find the amplitude and the smallest positive time at which y is a maximum.
  5. Express cos x − sin x in the form R cos(x + α), R > 0, 0 < α < π/2. Hence solve cos x − sin x = 0.5 for −π ≤ x ≤ π.

Quick answers: Q1: R = 13, α = arctan(12/5) ≈ 1.176 rad. So 13 sin(x − 1.176). Q2: R = √(36+64) = 10. Max 10 at x = π/2 − arctan(6/8) = π/2 − 0.6435 ≈ 0.927. Min −10 at x = 0.927 + π ≈ 4.069. Q3: R = 2, α = π/6. 2 sin(x + π/6) = 1, sin(x + π/6) = 0.5, x + π/6 = π/6 or 5π/6. So x = 0 or x = 2π/3. Q4: R = 5 metres. y = 5 sin(2t + 0.6435). Max when 2t + 0.6435 = π/2, so t = (π/2 − 0.6435)/2 ≈ 0.464 s. Q5: R = √2, α = π/4. √2 cos(x + π/4) = 0.5, cos(x + π/4) = 1/(2√2) ≈ 0.354, x + π/4 = ±1.209. So x ≈ 0.424 or x ≈ −2.00.

Two-minute summary

  • Small angle approx (x in radians, x small): sin x ≈ x, cos x ≈ 1 − x²/2, tan x ≈ x.
  • Applications: evaluating limits as x→0, simplifying expressions involving small angles, physics modelling.
  • R sin(x+α) form: R = √(a²+b²), tan α = b/a (or appropriate ratio for chosen form).
  • Uses: max/min of a sin + b cos (= ±R); solving a sin x + b cos x = c; modelling waves with phase.
  • Always check your α is in the right quadrant based on the signs of a and b and the chosen form.
  • When solving: remember sin θ = k has two solutions per 2π — arcsin(k) and π − arcsin(k).

Retrieval test

  2. Write the general identity: a sin x + b cos x ≡ ? (give R and α formulae).
