Hyperbolics [RED × 2]

H2 Inverse hyperbolics · H3 Hyperbolic calculus — paired session

Session plan — two RED topics in one session
  1. 10 min — Recap H1 (basic hyperbolics): definitions, identities.
  2. 30 min — H2 Inverse hyperbolics: log forms, domains, derivatives.
  3. 35 min — H3 Hyperbolic calculus: differentiation, integration, standard integrals.
  4. 20 min — Worked examples combining both topics.
  5. 15 min — Practice set.

Strategy. Hyperbolic functions are almost isomorphic to trigonometric functions. Learn the mapping (sin ↔ sinh, cos ↔ cosh, tan ↔ tanh) and 80% of what you need falls out. Key differences: sign flips in identities, no periodicity.

Recap — basic hyperbolics (H1)

Definitions

sinh x = (ex − e−x) / 2   ·   cosh x = (ex + e−x) / 2   ·   tanh x = sinh x / cosh x

Core identity (the one to memorise)

cosh2 x − sinh2 x = 1

Note the minus sign — the trigonometric analogue is cos²x + sin²x = 1. This is the key difference.

Derived identities (worth knowing)

sinh(2x) = 2 sinh x cosh x
cosh(2x) = cosh²x + sinh²x = 2cosh²x − 1 = 1 + 2sinh²x

Graph shapes — know these cold

sinh x: odd function, passes through origin, unbounded. cosh x: even, minimum value 1 at x=0, U-shaped (catenary). tanh x: bounded between ±1, S-curve through origin.

H2 — Inverse hyperbolics

Log forms — you must memorise these

Inverse hyperbolic functions have closed-form logarithmic expressions (unlike inverse trig). Derive them by solving y = sinh x etc. for x. These are in the formula booklet but you need them fluently.

arsinh x = ln(x + √(x² + 1))   (for all real x)

arcosh x = ln(x + √(x² − 1))   (for x ≥ 1, principal branch)

artanh x = ½ ln((1 + x)/(1 − x))   (for |x| < 1)

Derivation of arsinh (for practice)

Start with: y = sinh x, so x = arsinh y. Rearrange for x.

y = (ex − e−x) / 2

2y = ex − e−x

Multiply through by ex:   2y·ex = e2x − 1

(ex)² − 2y·(ex) − 1 = 0    (quadratic in ex)

ex = (2y ± √(4y² + 4)) / 2 = y ± √(y² + 1)

Since ex > 0, take the + root:   ex = y + √(y² + 1)

Therefore x = ln(y + √(y² + 1)), so arsinh y = ln(y + √(y² + 1)). ✓

Exam use. If asked to "express arsinh(3) as a natural logarithm in exact form", just plug into the formula: arsinh(3) = ln(3 + √10). That's the exam answer — no decimal approximation.

Derivatives of inverse hyperbolics

d/dx [arsinh x] = 1 / √(x² + 1)

d/dx [arcosh x] = 1 / √(x² − 1)   (for x > 1)

d/dx [artanh x] = 1 / (1 − x²)   (for |x| < 1)

Pattern: these mirror the inverse trig derivatives 1/√(1−x²), 1/√(1+x²)... but with signs flipped. Memorise as a set; they appear routinely in integration by recognition.

H3 — Hyperbolic calculus

Standard derivatives

d/dx [sinh x] = cosh x   (same as trig, no sign change)

d/dx [cosh x] = sinh x   (differs from trig: no minus)

d/dx [tanh x] = sech2 x = 1 − tanh2 x
The critical trap. d/dx [cos x] = −sin x, BUT d/dx [cosh x] = +sinh x. Many students apply the trig minus sign by habit. Don't.

Standard integrals — recognition targets

The whole point of learning inverse hyperbolics is to recognise integrals whose answer is an inverse hyperbolic function.

∫ 1 / √(x² + a²) dx = arsinh(x/a) + c

∫ 1 / √(x² − a²) dx = arcosh(x/a) + c   (x > a)

∫ 1 / (a² − x²) dx = (1/a) artanh(x/a) + c   (|x| < a)

Comparison with inverse trig integrals — you already know these from A-level Maths:

  • ∫ 1/√(a² − x²) dx = arcsin(x/a) + c  (minus sign under root → arcsin)
  • ∫ 1/(a² + x²) dx = (1/a) arctan(x/a) + c  (plus sign in denom → arctan)

Pattern: look at the sign inside the surd or denominator to decide whether it's a trig or hyperbolic inverse.

Worked example 1 — inverse hyperbolic in log form

Q: Express arcosh(5/3) in the form ln(p/q) where p and q are integers.

arcosh(5/3) = ln(5/3 + √((5/3)² − 1))

= ln(5/3 + √(25/9 − 9/9))

= ln(5/3 + √(16/9))

= ln(5/3 + 4/3)

= ln(9/3) = ln 3

Worked example 2 — integration by recognition

Q: Evaluate ∫ 1/√(x² + 9) dx from 0 to 4.

Recognition: 1/√(x² + 9) = 1/√(x² + 3²). This is the arsinh(x/a) form with a = 3.

So ∫ = [arsinh(x/3)] from 0 to 4

= arsinh(4/3) − arsinh(0)

= ln(4/3 + √((4/3)² + 1)) − 0

= ln(4/3 + √(16/9 + 9/9))

= ln(4/3 + √(25/9))

= ln(4/3 + 5/3) = ln 3

Worked example 3 — substitution to hyperbolic

Q: Find ∫ √(x² + 4) dx.

Method. Try substitution x = 2 sinh u (because x² + 4 suggests sinh substitution with a = 2). Then dx = 2 cosh u du and x² + 4 = 4sinh²u + 4 = 4(sinh²u + 1) = 4cosh²u.

So √(x² + 4) = 2 cosh u.

∫ √(x² + 4) dx = ∫ 2 cosh u · 2 cosh u du = 4 ∫ cosh²u du

Use the identity cosh²u = (cosh(2u) + 1)/2:

= 4 · ½ ∫ (cosh(2u) + 1) du

= 2 [½ sinh(2u) + u] + c

= sinh(2u) + 2u + c

= 2 sinh u · cosh u + 2u + c   (double-angle)

Back-substitute: sinh u = x/2, cosh u = √(x² + 4)/2, u = arsinh(x/2).

= 2 · (x/2) · √(x² + 4)/2 + 2 arsinh(x/2) + c

= (x/2)·√(x² + 4) + 2 arsinh(x/2) + c

Two-minute summary

  • cosh² x − sinh² x = 1 (note the minus — differs from trig).
  • d/dx cosh x = +sinh x (NOT minus — trap).
  • Inverse log forms: arsinh = ln(x + √(x²+1)); arcosh = ln(x + √(x²−1)); artanh = ½ ln((1+x)/(1−x)).
  • Standard integrals: ∫dx/√(x²+a²) = arsinh(x/a); ∫dx/√(x²−a²) = arcosh(x/a); ∫dx/(a²−x²) = (1/a) artanh(x/a).
  • Sign check: signs under the root/in the denominator determine trig vs hyperbolic inverse.
  • Hard integrals with √(x²±a²): substitute x = a sinh u or x = a cosh u.

Practice problems

  1. Prove cosh² x − sinh² x = 1 from the exponential definitions.
  2. Express arsinh(√3) in exact log form.
  3. Express arcosh(13/12) in the form ln(p/q).
  4. Evaluate artanh(1/2) in exact log form.
  5. Differentiate y = cosh(3x² + 1).
  6. Differentiate y = arsinh(x²).
  7. Find ∫ 1/√(x² + 25) dx.
  8. Find ∫ 1/√(4x² − 9) dx. (Hint: factor out the 4 first.)
  9. Find ∫ 1/(16 − 9x²) dx.
  10. Find ∫ √(x² + 16) dx using substitution x = 4 sinh u.

Quick answers: Q2: ln(√3 + 2). Q3: ln(3/2). Q4: ½ ln 3. Q5: 6x·sinh(3x² + 1). Q6: 2x/√(x⁴ + 1). Q7: arsinh(x/5) + c. Q8: ½ arcosh(2x/3) + c. Q9: (1/12) artanh(3x/4) + c. Q10: (x/2)√(x² + 16) + 8 arsinh(x/4) + c.
